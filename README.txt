OVERVIEW
--------------------------------------------------------------------------------
This repository contains all artefacts produced in the above study, which
presents the first systematic three-way comparison of DCGAN, VAE-GAN, and
VAE-Diffusion architectures for oesophageal endoscopy image synthesis.

All three models were trained on 672 clinical images drawn from the publicly
available GastroVision dataset, covering four oesophageal classes:
  - Barrett's oesophagus
  - Oesophagitis
  - Gastro-oesophageal junction
  - Normal Z-line

IMPORTANT: The original GastroVision images are NOT included in this deposit.
They must be obtained directly from the GastroVision authors (see SOURCE
DATASET section below). Only the list of image identifiers used in this study
is provided here to ensure full reproducibility.

--------------------------------------------------------------------------------
REPOSITORY CONTENTS
--------------------------------------------------------------------------------

1. subset_index/
   └── gastrovision_672_ids.csv
       The complete list of 672 GastroVision image filenames/IDs used for
       training and evaluation, organised by class:
         - Barrett's oesophagus : 168 images
         - Oesophagitis         : 168 images
         - GE Junction          : 168 images
         - Normal Z-line        : 168 images
       Column headers: [filename, class_label, split]

2. model_weights/
   ├── dcgan_epoch340.pth
   │     DCGAN generator weights at best FID checkpoint (epoch 340/500).
   │     Architecture: 100-dim noise → 128×128×3 RGB output.
   │     FID = 281.08 | SSIM = 0.231 | PSNR = 10.48 dB
   │
   ├── vaegan_epoch500.pth
   │     VAE-GAN encoder + decoder weights at epoch 500.
   │     Latent dimension: 128. Input/output: 64×64×3.
   │     FID = 132.99 | SSIM = 0.343 | PSNR = 13.32 dB
   │
   └── vaediffusion_epoch300.pth
         VAE-Diffusion full model weights (VAE + latent U-Net denoiser)
         at best FID checkpoint (epoch 300/500).
         Latent shape: 4×8×8. Input/output: 64×64×3.
         FID = 121.59 | SSIM = 0.387 | PSNR = 13.14 dB

3. synthetic_images/
   ├── dcgan_500_samples/
   │     500 synthetic oesophageal endoscopy images generated by DCGAN
   │     at epoch 340. Resolution: 128×128 pixels. Format: PNG.
   │
   ├── vaegan_500_samples/
   │     500 synthetic images generated by VAE-GAN at epoch 500.
   │     Resolution: 64×64 pixels. Format: PNG.
   │     Generated by sampling z ~ N(0,I) through the trained decoder.
   │
   └── vaediffusion_500_samples/
         500 synthetic images generated by VAE-Diffusion at epoch 300.
         Resolution: 64×64 pixels. Format: PNG.
         Generated using full 1000-step DDPM reverse diffusion sampling.

   NOTE: All 1,500 images were used for FID evaluation against a 500-image
   real reference set drawn from GastroVision under identical preprocessing.

4. code/
   ├── train_dcgan.py          Training script for DCGAN
   ├── train_vaegan.py         Training script for VAE-GAN
   ├── train_vaediffusion.py   Training script for VAE-Diffusion (2-phase)
   ├── evaluate_metrics.py     Unified FID / SSIM / PSNR evaluation pipeline
   ├── generate_samples.py     Inference script for all three models
   └── requirements.txt        Python package dependencies

--------------------------------------------------------------------------------
REQUIREMENTS & ENVIRONMENT
--------------------------------------------------------------------------------
  Python        : 3.8+
  PyTorch       : 1.12+
  TensorFlow    : 2.9+  (used for DCGAN implementation)
  pytorch-fid   : 0.3.0 (FID computation with InceptionV3 features)
  scikit-image  : 0.19+ (SSIM and PSNR computation)
  CUDA          : 11.3+ (training performed on NVIDIA Quadro RTX 6000)

Install all dependencies:
  pip install -r code/requirements.txt

--------------------------------------------------------------------------------
HOW TO REPRODUCE THE RESULTS
--------------------------------------------------------------------------------

Step 1 — Obtain the GastroVision dataset
  Download from: https://github.com/DebeshJha/GastroVision
  Place images in: ./data/gastrovision/

Step 2 — Prepare the oesophageal subset
  Use the provided index file to extract the 672 images:
    python code/generate_samples.py --index subset_index/gastrovision_672_ids.csv
                                    --data_dir ./data/gastrovision/
                                    --output_dir ./data/oesophageal_subset/

Step 3 — Train the models (or skip to Step 4 to use pretrained weights)
    python code/train_dcgan.py          --data_dir ./data/oesophageal_subset/
    python code/train_vaegan.py         --data_dir ./data/oesophageal_subset/
    python code/train_vaediffusion.py   --data_dir ./data/oesophageal_subset/

Step 4 — Generate synthetic images using pretrained weights
    python code/generate_samples.py \
        --dcgan_weights    model_weights/dcgan_epoch340.pth \
        --vaegan_weights   model_weights/vaegan_epoch500.pth \
        --vaediff_weights  model_weights/vaediffusion_epoch300.pth \
        --output_dir       ./generated/

Step 5 — Evaluate FID, SSIM, and PSNR
    python code/evaluate_metrics.py \
        --real_dir      ./data/oesophageal_subset/ \
        --generated_dir ./generated/ \
        --n_samples     500

Expected output (should match Table 4 in the paper):
  Model           FID      SSIM    PSNR (dB)
  DCGAN           281.08   0.231   10.48
  VAE-GAN         132.99   0.343   13.32
  VAE-Diffusion   121.59   0.387   13.14
